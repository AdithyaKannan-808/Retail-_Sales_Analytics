# Retail-_Sales_Analytics
Built an interactive Excel Business Dashboard to analyze sales, profit, returns, and segment performance from 2018–2021. The dashboard provides a clear, consolidated view of trends, helping management make faster, data-driven decisions on product strategy and sales optimization.

Dashboard Objectives

Convert large transactional data into an interactive decision-making tool.

Track KPIs: Sales, Profit, Orders, Return Rate, Segment Margin, Region Margin, and Top Performer.

Identify top-performing products, salespersons, regions, and buyer segments.

Reveal profitability patterns across time, categories, and customer types.

Enable management to apply dynamic slicers for real-time insights.
